Apresentação coorporativa

2018

Realizar apresentações para grandes públicos é sempre um desafio, principalmente quando se trata de convencê-los a tomar determinada ação. Pensando nisso, este projeto propôs uma apresentação que conta dados e fatos através de uma história. Ilustrado com fotografias, uma boa diagramação e um toque de criatividade, os slides dão vida a uma ideia.
